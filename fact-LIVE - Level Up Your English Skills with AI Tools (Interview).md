จากการตรวจสอบพยานหลักฐานเชิงประจักษ์ในสื่อนำเสนอ พบพฤติการณ์จงใจบิดเบือนสถานะทางวิชาชีพผ่านการสื่อสารที่สร้างความคลุมเครืออย่างมีนัยสำคัญ<br> โดยเฉพาะการใช้ข้อความ "Microsoft MVP Program Transition" ซึ่งตรวจสอบแล้วพบว่าเป็นตำแหน่งที่ไม่มีการรับรองจริงในสาระบบของ Microsoft แต่เป็นการสร้างคำนิยามขึ้นเองเพื่อสร้าง "ความน่าเชื่อถือเทียม" False Authority ในขณะที่ชื่อของเขายังไม่ปรากฏในทำเนียบ MVP Global Directory การกระทำนี้ถือเป็นการอาศัยตราสัญลักษณ์ในอดีตมาเป็นเครื่องมือในการแสวงหาการยอมรับในระดับที่ตนเองยังไปไม่ถึง เข้าข่ายการหลอกลวงด้วยความจริงบางส่วน (Half-truth Deception) เพื่อหวังผลในการสร้างภาพลักษณ์ (Social Capital) บนสื่อสังคมออนไลน์
นอกจากนี้ การอ้างตนเป็น "Freelance AI Engineer" และ "Solution Engineer" ในขณะที่สถานะหลักตามพยานหลักฐานระบุว่าเป็นเพียงนักศึกษาปริญญาตรี (Undergraduate) ถือเป็นพฤติกรรมการยกระดับตนเองเกินจริง (Title Inflation) ที่ขัดต่อหลักจริยธรรมวิชาชีพไอทีอย่างร้ายแรง เนื่องจากคำว่า "วิศวกร" (Engineer) ควรสงวนไว้สำหรับผู้ที่มีคุณวุฒิและประสบการณ์ระดับอุตสาหกรรมที่ตรวจสอบได้ พฤติกรรมดังกล่าวสะท้อนถึงเจตนาในการปั่นโปรไฟล์ (Profile Inflating) เพื่อสร้างทางลัดสู่ความสำเร็จโดยปราศจากฐานรากของคุณวุฒิที่แท้จริง การนำเสนอข้อมูลที่ปั้นแต่งเช่นนี้ไม่เพียงแต่เป็นการทำลายบรรทัดฐานของชุมชนนักพัฒนา แต่ยังเป็นความเสี่ยงต่อผู้ว่าจ้างหรือผู้ที่หลงเชื่อในข้อมูลบิดเบือนดังกล่าว <br>จึงขอบันทึกไว้เป็นหลักฐานใน Repository แห่งนี้เพื่อการเฝ้าระวังทางสังคมสืบไป
